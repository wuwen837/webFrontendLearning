CSS(层叠样式表)

CSS即层叠样式表 ，是网页样式设计的核心语言，与HTML、JavaScript并称前端三剑客。用于定义网页元素的布局、颜色、字体等视觉样式，支持层叠、继承特性，能实现结构与样式分离，让网页设计更灵活、维护更便捷。

css时开放网络核心语言之一。 CSS由哈肯·维姆·莱1994年提出，1996年W3C发布CSS1.0正式规范，实现网页结构与样式分离，支持基础字体、边距等样式。1998年CSS2.0上线，新增浮动、定位、媒体查询等布局功能，2001年优化为CSS2.1，成为长期主流版本。2011年起进入CSS3时代，采用模块化发展，陆续推出Flexbox、Grid、变量、动画等特性，适配现代网页的复杂布局与多端需求，至今仍在持续迭代更新。

  • CSS 诞生于 1994 年，1996 年发布首个正式版本 CSS 1.0，核心目标是分离 HTML 结构与样式；
  • CSS 2.1 是长期主流版本，完善了布局能力；CSS 3 采用模块化发展，新增大量现代样式特性；
  • 如今 CSS 仍在持续迭代，重点适配移动端 、复杂布局等现代网页开发场景。
